# Contributing

Vielen Dank fuer Ihr Interesse, zur Sunbooster-Home-Assistant-Integration beizutragen!

## Bugs und Feature-Requests

Bitte erstellen Sie ein GitHub-Issue unter https://github.com/sunboostergmbh/ha-sunbooster-powerstation/issues. Nutzen Sie die vorhandenen Templates.

## Pull Requests

Aktuell befindet sich das Projekt im **internen Beta-Stadium**. Pull Requests von externen Entwicklern werden noch nicht angenommen, da die Architektur noch in Bewegung ist.

Sobald wir aus der Beta in v1.x stable wechseln, oeffnen wir dieses Repository auch fuer Community-PRs. Bis dahin: bitte Vorschlaege als Issue einreichen.

## Code of Conduct

Seien Sie freundlich, geduldig und hilfsbereit. Persoenliche Angriffe, Diskriminierung oder Trolling sind nicht akzeptabel.
