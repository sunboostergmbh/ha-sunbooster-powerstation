# Support

## Hilfe bekommen

- **GitHub Issues**: https://github.com/sunboostergmbh/ha-sunbooster-powerstation/issues  
  Fuer Bugs und Feature-Requests
- **E-Mail**: service@sunbooster.com  
  Fuer individuelle Fragen, Account- oder Key-Probleme

## Bevor Sie ein Issue erstellen

Pruefen Sie bitte:

- Sind Home Assistant und HACS auf dem neuesten Stand?
- Haben Sie die Integration auf die neueste Version aktualisiert?
- Steht in den Logs (Einstellungen -> System -> Logs) etwas zu `sunbooster`?
- Existiert das Issue bereits unter https://github.com/sunboostergmbh/ha-sunbooster-powerstation/issues\?

## Antwortzeiten

Wir bemuehen uns, E-Mails an service@sunbooster.com werktags innerhalb von 24-48 Stunden zu beantworten. GitHub-Issues werden im selben Zeitraum gesichtet.

Da die Integration aktuell im Beta-Stadium ist, gibt es keinen festen SLA.
